Installation

Drop a single CSS file into your project. No build step, no JavaScript, no dependencies.

Usage

Link the stylesheet and write semantic HTML. Most elements look great with zero classes.

Dark Mode

Automatic via prefers-color-scheme: dark, or toggle manually with a .dark class on <html>.

Themes

Apply a theme class to any container to restyle everything inside it.

Buttons

Buttons come in color variants, outline/ghost styles, and multiple sizes. Combine classes freely.

Cards

Flexible content containers with optional image, body, title, and action areas.

Badges

Small status indicators in every color variant, with outline and size options.

Alerts

Contextual feedback messages with colored left borders.

Tabs

Navigation tabs with an active state indicator. Add .tab-active to the current tab.

Toggles

iOS-style toggle switches built on native checkboxes. Zero JavaScript required.

Dropdown

Dropdown menus using native <details>/<summary>. No JS needed.

Modal

Modal dialogs using the native <dialog> element with backdrop blur.

Progress

Simple progress bars with color variants for different states.

Avatars

Circular image avatars in four sizes: sm, default, lg, and xl.

Tooltips

Pure-CSS tooltips via data-tooltip attribute. No JavaScript.

Inputs

Text inputs, selects, textareas, and labels with focus ring styling.

Input Groups

Combine inputs with prefix/suffix text or icons.

Container

Centered max-width container (1200px) with horizontal padding.

Grid

CSS Grid utilities for 2, 3, 4, and auto-fit column layouts. Responsive by default.

Flex Utilities

Row, column, alignment, and distribution helpers for flexbox layouts.

Spacing

Gap and padding utilities in five sizes: xs, sm, md, lg, xl.
